#  Diabetes Prediction using Machine Learning

Predicting diabetes in patients using the PIMA Indian dataset.

##  Dataset
- Target: `Outcome`
- Features: Glucose, BMI, Age, Insulin, etc.

##  Libraries
- Pandas, Scikit-learn, Matplotlib, SMOTE

##  Evaluation
- Accuracy, AUC, ROC Curve
